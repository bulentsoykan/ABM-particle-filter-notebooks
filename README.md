## What You Will Learn

### Module 1 — ABM Uncertainty
- What makes an ABM stochastic (the random dodge-left/right collision rule)
- How small random choices compound into large state uncertainty over time
- Why real-time data assimilation is necessary even with a perfect model

### Module 2 — Particle Filter Core Concepts
- The three-step PF cycle: **Predict → Reweight → Resample**
- How particles represent a probability distribution over possible states
- Systematic resampling algorithm (O(N), low variance)
- Interactive animation tracking a nonlinear moving object

### Module 3 — PF Applied to an ABM
- The "identical twin" experimental design from the paper
- Each particle as a full ABM simulation (not just a number)
- How the PF constrains the ensemble towards the pseudo-truth
- Effect of N_agents and N_particles on tracking error

### Module 4 — Particle Degeneracy
- Weight collapse: how ESS drops to 1 without resampling
- Particle deprivation after resampling (all clones)
- Jitter/roughening to restore diversity
- Tuning the noise parameter sigma_p

### Module 5 — Curse of Dimensionality
- Why collisions grow O(N^1.7) with agent count
- Error heatmap over (N_agents, N_particles) 
- Why the Kalman Filter fails for ABMs (non-linear, non-Gaussian state distribution)
